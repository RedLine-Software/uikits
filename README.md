# UIKits2 

A complete UIs for mobile and tablet, which include 16 categories.

 * Start
 * SignUp & Login
 * Walkthrough
 * Loading
 * Profiles
 * Feed
 * Article
 * Activity
 * Create
 * Chat
 * Contact
 * Store & Shop
 * Shopping Carts & Checkout
 * Navigation
 * Photos
 * Dashboard
